# Digital-Maturity-Assessment

## Methodology

### Digital Maturity Framework
- Defines six dimensions: *Strategy, Customer Experience, Technology, Data & AI, People & Culture, Operations*  
- Each contains multiple sub-areas with default weights.

### Survey Layer
- Auto-generated survey questions aligned to all sub-areas  
- Converts responses into a long-format dataset

### Scoring Engine
- Computes weighted scores  
- Aggregates to dimension-level maturity  
- Classifies into *Initial, Developing, Established, Advanced*

### Visual Analytics
- Heatmaps  
- Radar charts  
- Dimension-wise contribution graphs  

### Roadmap Engine
- Priority labels: *Critical, Improve, Healthy*  
- Actionable recommendations for each sub-area
